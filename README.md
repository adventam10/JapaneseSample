# JapaneseSample
日本語コードでiOSアプリを作成する！！

## 環境
* Xcode 11.0
* swift 5.1
* 最小ターゲット iOS 13.0

## アプリ機能
検索バーに入力した文字でQiitaの記事を検索し、記事一覧を表示するアプリ。記事タップで外部ブラウザで記事を表示する。

[簡単な画面仕様](https://github.com/adventam10/JapaneseSample/blob/master/画面仕様.md)

## API
[Qitia APIを使用](https://qiita.com/api/v2/docs#get-apiv2items)
